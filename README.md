# 學習Git的紀錄
## 1. Git初始設定
安裝Git(debian)
```
$ apt-get install git
```
### 讓 Git 知道這台電腦做的修改要連結到哪一個使用者
設定使用者名稱
```
$ git config --global user.name "<Your Name>"
```
設定電子郵件信箱
```
$ git config --global user.email "<your@gmail.com>"
```
